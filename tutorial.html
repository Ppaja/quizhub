Getting Started

1. Before we start

First, please tell me your GitHub Username, so I can add you to the GitHub project

2. Setup

after I added you to the GitHub project you can setup the project

download the following programs:

  • GitHub Desktop
  • VS Code

3. Download the project

Now that you have all the programs, you can download the project with GitHub Desktop

open GitHub Desktop and click "clone repository from the internet" and click the "URL" tab.

navigate to the URL tab and paste the URL of the project

https://github.com/Ppaja/quizhub


4. set up your branch

after you cloned the project create your own branch

click on the "Branches" tab and "New branch" and name it for example like your username

a branch is a copy of the project that is separate from the main branch. Whatever you add or do will not affect the main branch until you manually merge it

after you created your own branch click the "Push" button

5. start coding

now you can click "open with VS Code" and start coding

I would recommend you read the rest of this tutorial to get a better understanding of the project and how the code works

click here to view how the code works, whats good to know etc.

6. commit changes and push to your branch

after you added, changed or deleted something and come back to GitHub Desktop you can see the changes you made

for each file type a small description of what you did and click "commit changes"

do this for all files.

then click "Push origin"

now your changes are pushed to your GitHub page

but not into the main code/branch, just into your branch

7. create pull request

To submit your changes to the main code/branch, click "preview Pull request"

you get a overview of all the changes you made

click "create pull request"

now a browser opens and you have to descripe a general description of all your changes and press the final "create pull request" button

And thats it! You now have to wait until another author approves your changes and merge your pull request into the main code/branch

FAQ

What is QuizHub?

QuizHub is a web-based project that focuses on creating an interactive platform for hosting and playing quizzes.

Its mostly about learning how to use GitHub and code a project

What can I do?

You can add/modify/delete whatever you want. There is no right or wrong, do what you think would be cool :)

Its mostly about learning how to use GitHub and code a project

Examples of what you can do?

How about creating a own quiz about something? Create a new file, for example "history.html", use the template code to create your own quiz.

After you done you can link it to main page or somebody else can do it

What if I mess up?

No worries, its all about learning how to use GitHub and code a project. You cant do anything wrong. The main branch is protected, so you cant accidentally mess up!
